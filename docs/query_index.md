## Difficulty Levels

- **Beginner**: Basic SELECT, WHERE, GROUP BY, simple JOINs
- **Intermediate**: Multiple JOINs, subqueries, CASE statements, window functions basics
- **Advanced**: CTEs, complex window functions, self-joins, advanced aggregations

## SQL Concepts Covered

### Basic Concepts
- SELECT, FROM, WHERE, ORDER BY
- GROUP BY, HAVING
- INNER JOIN, LEFT JOIN
- Aggregate functions (SUM, COUNT, AVG, MIN, MAX)

### Intermediate Concepts
- CASE statements
- Multiple table JOINs
- Subqueries
- UNION / UNION ALL
- Date functions (strftime)
- COALESCE, NULLIF

### Advanced Concepts
- Common Table Expressions (CTEs)
- Window Functions (ROW_NUMBER, RANK, LAG, LEAD, NTILE)
- Running totals (SUM OVER)
- Moving averages (AVG OVER with ROWS BETWEEN)
- Self-joins
- Complex aggregations
